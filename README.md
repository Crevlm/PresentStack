# Present Stack

A frantic 3D stacking game made for the 12 Seconds of Christmas Game Jam.

## About
You have exactly 12 seconds to build the tallest tower of presents possible. Grab boxes, stack them high, and pray your tower doesn't topple before time runs out! Higher the tower, higher the score!
